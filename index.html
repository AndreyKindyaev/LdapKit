    LDAP Kit Reference

    LDAP Kit is a framework which provides an Objective-C interface for the OpenLDAP client libraries on Max OS X and iOS. LDAP Kit manages the conversion between C data types and Objective-C classes, manages background threads for executing LDAP tasks, and manages memory allocated by OpenLDAP via NSAutoreleasePool objects.

    The LDAP Kit Project Information page contains basic instructions for integrating LDAP Kit into iOS and Mac OS X applications. The source code for LDAP Kit and this documentation are available for download from Github.
